# Netflix Content Catalog Analysis

An exploration of ~8,800 Netflix titles: content mix, growth, geography, genres, and target audience.

**Key findings:**
- The catalog is ~70% Movies, 30% TV Shows, and grew explosively from 2016 onward.
- Content is globally sourced but US-led; International content and Dramas are the top genres.
- The library skews toward mature audiences (TV-MA + TV-14 dominate).
- 67% of TV shows have just one season — Netflix greenlights many series but renews selectively.

**Tools:** Python, pandas, matplotlib (includes handling of messy real-world data: missing values, mixed-unit duration fields, multi-value genre cells)

**Data:** [Netflix Movies and TV Shows (Kaggle)](https://www.kaggle.com/datasets/shivamb/netflix-shows) — download `netflix_titles.csv` into this folder, then run the notebook top to bottom.
